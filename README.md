# joystick-inversion

## Installation

Clone the repository

```bash
git clone <thingy>
```

Create an activate a virtual environment in the project root

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install using

```bash
pip install -e .
```

You can train and evaluate the model by running

```bash
python scripts/run.py
```
